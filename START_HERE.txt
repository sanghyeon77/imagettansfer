🎯 목표: 영상을 자동으로 캡처해서 디스코드에 업로드하기

============================================================
⚡ 가장 빠른 방법 (3단계)
============================================================

1. install.bat 더블클릭
   └─> 모든 필요한 것들을 자동으로 설치

2. start_server.bat 더블클릭
   └─> 서버 시작

3. 브라우저에서 http://127.0.0.1:5000 접속
   └─> 웹 페이지에서 영상 업로드


============================================================
📋 필요한 것들 (자동으로 설치됨)
============================================================

✅ Python 3.11 이상
✅ Flask (웹 서버)
✅ moviepy (영상 처리)
✅ discord-webhook (디스코드 업로드)
✅ FFMPEG (영상 처리 도구)

============================================================
❓ 설치가 안 되나요?
============================================================

1. Python이 없다고 나오면:
   → https://www.python.org/downloads/
   → "Add Python to PATH" 체크 필수!
   → 설치 후 컴퓨터 재시작

2. FFMPEG이 없다고 나오면:
   → PowerShell 관리자 권한으로 실행
   → choco install ffmpeg
   → 또는 "설치방법.txt" 파일 참고

3. 그래도 안 되면:
   → "설치방법.txt" 파일을 열어보세요
   → 또는 "INSTALL_GUIDE.md" 파일 참고


============================================================
📁 중요한 파일들
============================================================

install.bat           ← 이것 먼저 실행 (설치)
start_server.bat      ← 그 다음 이것 실행 (서버 시작)
설치방법.txt          ← 문제 생기면 이것 읽기
app.py                ← 서버 프로그램
templates/index.html  ← 웹 페이지


============================================================
🎬 사용 예시
============================================================

1. start_server.bat 실행
2. 브라우저에서 http://127.0.0.1:5000 접속
3. 영상 파일 선택 (예: movie.mp4)
4. 웹훅 URL 붙여넣기 (위에 있는 URL)
5. 캡처 주기 입력 (예: 10분)
6. "시작" 버튼 클릭
7. 디스코드에서 자동으로 스크린샷 확인!


============================================================
💡 팁
============================================================

- 10분 주기 = 영상의 0분, 10분, 20분... 시점을 캡처
- 서버를 중지하려면 Ctrl+C 누르기
- 웹 페이지에서 "중지" 버튼으로도 중지 가능
- 영상 끝에 도달하면 자동으로 종료됨
